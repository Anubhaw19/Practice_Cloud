# Practice_Cloud



## ssh 

* create a public and private ssh-keys in local host 

```bash
  ssh-keygen -t rsa
```

* copy the public-key to remote host for authentication
```bash
  ssh-copy-id user@ip
```
* ssh into a VM
```bash
  ssh user@ip
```
